# Study-Time: An Intelligent Study Assistant

Study-Time is a specialized AI application designed to help students optimize their learning routines and master core academic concepts. Built using **Python** and **Streamlit**, the application interfaces with advanced Large Language Models (LLMs) to provide customized feedback, active recall testing, and targeted academic guidance.

## Features
* **Adaptive Evaluation:** Custom benchmark questions testing academic accuracy.
* **Streamlit UI:** An intuitive, user-friendly interface built for high student engagement.
* **Multi-Model Support:** Configured to dynamically process queries using state-of-the-art LLM architectures.

## Technical Stack
* **Language:** Python 3.11
* **Framework:** Streamlit
* **Deployment:** Cloud Infrastructure
